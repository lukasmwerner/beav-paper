This project is meant to make abstract data structure behavior easier to see.

## Why it exists

A lot of CS topics make more sense once you can watch the state change step by step.

## Current scope

Right now it includes a few core structures and common operations.

## Next steps

I want to add algorithm walkthroughs, speed controls, and better annotations for each step.
