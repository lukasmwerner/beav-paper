I built this project after planning too many terms with a spreadsheet that kept getting out of sync.

## Main features

- compare multiple schedule options
- add work shifts and club meetings
- estimate open study blocks
- save several term plans

## Reflection

This project taught me a lot about edge cases in scheduling logic and how quickly simple planning tools become real product design problems.
